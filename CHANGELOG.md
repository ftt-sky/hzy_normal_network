## [0.0.1] - 2022-11-08

### 新增
- 初始版本发布
- 基于 Dio 的网络请求封装
- 统一的响应数据格式 (HzyNormalResponse)
- 灵活的配置选项 (HzyNormalHttpConfig)
- 自定义数据转换器支持 (HzyNormalTransFormer)
- 完善的异常处理机制
- 请求/响应日志支持
- 支持常见的 HTTP 方法 (GET, POST, PUT, PATCH, DELETE)
- 支持文件上传和下载
- 支持请求取消
- 支持进度监听

### 功能特性
- 🚀 基于 Dio 的高性能网络请求

- 🎯 支持自定义数据转换器
- ⚡ 支持请求拦截器
